# GitHub contents for typed synthetic neuroangiography benchmark v0.1

This repository carries reproducible source code and manuscript source only. Generated data artifacts are archived on Zenodo.

Keep in GitHub:
- `scripts/` — generator, baseline, DIAS adapter, verification, figure, and packaging scripts.
- `tests/` — unit tests and artifact validation tests.
- `schemas/` — schema definitions mirrored from the Zenodo archive for validation.
- `paper/` — manuscript source and bibliography.
- `README.md`, `RELEASE_v0.1.md`, `CITATION.cff`, `NOTICE`, `LICENSE`, `ZENODO_CONTENTS.md`, `GITHUB_CONTENTS.md`.

Do not keep generated/bulk artifacts in GitHub:
- `data/`
- `outputs/`
- `deliverables/`
- `zenodo_staging/`
- generated `.zip` archives
- generated PDFs/figure packs except where deliberately needed for small documentation previews
